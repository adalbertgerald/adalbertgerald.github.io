title: Non-native English speakers Learning Programming
description: This project focuses on understanding the barriers that non-native English speakers face while learning programming as the majority of the computing ecosystem (e.g., programming language documentation) is primarily in English. Once we identify these barriers, we will design and develop interventions/tools to help non-native English speakers overcome these barriers. In the past, we have taughtcomputer science using bilingual techniques like translanguaging and code-switching to non-native English speakers, so that they are provided equitable access to CS/programming.
